자료형#

type() : 자료형 확인#

  • type(자료) : 자료형을 확인

  • 자료형에는 다음과 같은 종류들이 존재합니다.

    • 실수 (float)

    • 정수 (int)

    • 문자 (str)

    • 불린 (Boolean)

    • 리스트 (list)

    • 딕셔너리 (dic)

정수형(int)과 실수형(float)#

# 정수형 자료 확인하기
print(type(10))

<class 'int'>

# 실수형 자료 확인하기
print(type(10.5))

<class 'float'>

# 정수 뒤에 .을 붙이면 실수형으로 변환됩니다.
print(type(10.))

<class 'float'>

  • aen : \(a \times 10^n\)

# 1 곱하기 10의 3제곱
x = 1e3

print(x)

1000.0

# 3 곱하기 10의 -2제곱
x = 3e-2

print(x)

0.03

  • abs() : 절댓값 함수

print(abs(-10))

10

print(abs(5.5))

5.5

문자형 (str)#

  • 문자형 (string)은 보통 "문자내용" 또는 '문자내용' 과 같이 생성합니다.

# 문자형 자료를 생성합니다.
x = "안녕하세요"

print(x)
print(type(x))

안녕하세요
<class 'str'>

# 문자형 자료를 생성합니다.
y = '반갑습니다'

print(y)
print(type(y))

반갑습니다
<class 'str'>

불린형 (boolean)#

  • 불린형 (Boolean)은 True 또는 False와 같이 생성합니다.

# 불린형 자료 True 를 생성합니다
ttt = True

print(ttt)
print(type(ttt))

True
<class 'bool'>

# 불린형 자료 False를 생성합니다.
fff = False

print(fff)
print(type(fff))

False
<class 'bool'>

리스트형 (list)#

  • 리스트형(list)은 인덱스 번호로 값을 호출하는 구조의 자료형입니다.

  • 리스트를 생성할 때는 [원소0, 원소1, 원소2, ...] 와 같이 생성합니다.

# 리스트를 생성합니다.
x = [10,20,30,40]

print(x)
print(type(x))

[10, 20, 30, 40]
<class 'list'>

  • 리스트의 원소를 호출할 때는 리스트명[인덱스번호]와 같이 호출합니다.

  • 리스트형의 인덱스는 정방향의 경우 0부터 1씩 증가하고 역방향의 경우 -1 부터 1씩 감소합니다.

# 리스트를 생성합니다.
x = [10,20,30,40]

# 인덱스를 이용하여 리스트의 원소를 호출합니다.
print(x[0])
print(x[1])
print(x[2])
print(x[3])

print(x[4])

  • len(리스트명) : 리스트의 원소 개수를 반환합니다.

x = ['a','b','c']

print(len(x))

3

  • 슬라이싱(slicing)은 리스트에서 특정 범위의 요소들을 추출하는 것을 말합니다.

x = ['a','b','c','d','e','f']

# 2번 인덱스 이상 추출
x[2:]

['c', 'd', 'e', 'f']

x = ['a','b','c','d','e','f']

# 4번 인덱스 미만 추출
x[:4]

['a', 'b', 'c', 'd']

x = ['a','b','c','d','e','f']

# 2번 인덱스 이상 4번 인덱스 미만 추출
x[2:4]

['c', 'd']

  • 리스트에 원소 추가 : 리스트명.append(원소)

  • 리스트에 원소 제거

딕셔너리형#

  • 딕셔너리형(dic)은 key 값으로 value 를 호출하는 구조의 자료입니다.

  • 딕셔너리를 생성할 때는 {key_A : value_A, key_B : value_B, …} 와 같이 생성합니다.

# 딕셔너리를 생성합니다.
x = {'a':1, 'b':2, 'c':3}

print(x)
print(type(x))

{'a': 1, 'b': 2, 'c': 3}
<class 'dict'>

  • 딕셔너리의 value를 호출할 때는 딕셔너리[key] 와 같이 호출합니다.

# 딕셔너리를 생성합니다.
x = {'a':1, 'b':2, 'c':3}

# 딕셔너리의 key 값으로 value를 호출합니다.
print(x['a'])
print(x['b'])
print(x['c'])

1
2
3

# 딕셔너리를 생성합니다.
x = {'a':1, 'b':2, 'c':3}

# 딕셔너리에 정의되지 않은 key를 호출하면 에러가 발생합니다.
print(x['d'])

---------------------------------------------------------------------------
KeyError                                  Traceback (most recent call last)
Cell In[16], line 5
      2 x = {'a':1, 'b':2, 'c':3}
      4 # 딕셔너리에 정의되지 않은 key를 호출하면 에러가 발생합니다.
----> 5 print(x['d'])

KeyError: 'd'

  • 시뮬레이션 환경으로부터 받아온 정보가 params에 딕셔너리로 저장되어 있습니다.

  • 보상함수는 params 변수를 입력받아 보상을 출력하는 구조입니다.

  • 예시

    params = {
    'x' : 10,  # 차량의 x 좌표
    'y' : 20,  # 차량의 y 좌표
    'speed' : 3.5,  # 차량의 속도
    'steering_angle' : 12.7,  # 모든 바퀴가 트랙 내부에 위치하는지 여부
    'all_wheels_on_track' : True,
    # 기타 등등

}

자료형 변환#

  • float(자료) : 실수형으로 변환

  • int(자료) : 정수형으로 변환

  • str(자료) : 문자형으로 변환

# 정수형을 실수형으로 변환
x = 10
y = float(x)

print(y)
print(type(y))

10.0
<class 'float'>

# 문자을 실수형으로 변환
x = '36.5'
y = float(x)

print(y)
print(type(y))

36.5
<class 'float'>

# 숫자가 아닌 문자를 실수형으로 변환하면 에러가 납니다.

x = 'hello'
y = float(x)

---------------------------------------------------------------------------
ValueError                                Traceback (most recent call last)
Cell In[19], line 4
      1 # 숫자가 아닌 문자를 실수형으로 변환하면 에러가 납니다.
      3 x = 'hello'
----> 4 y = float(x)

ValueError: could not convert string to float: 'hello'

# 불린형을 실수형으로 변환
x = True
y = float(y)

print(y)
print(type(y))

36.5
<class 'float'>

# 실수형을 정수형으로 변환
x = 1.6
y = int(x)

print(y)
print(type(y))

1
<class 'int'>

# 실수형을 정수형으로 변환
x = -1.4
y = int(x)

print(y)
print(type(y))

-1
<class 'int'>

  • int()함수는 단순히 소수점 자리를 모두 버립니다.

  • 만약, 반올림을 하고자 한다면 round() 함수를 이용합니다.

x = 1.6
y = round(x)

print(y)

2

# 반올림 자리수를 지정하여 반올림 가능합니다.
x = 1.6217

print(round(x))
print(round(x,1))
print(round(x,2))
print(round(x,3))

2
1.6
1.62
1.622
